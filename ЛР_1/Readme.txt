Приветствую!
При запуске необходимо в консоли ввести номер начальной вершины. 
После будет выведен результат.

Например, для вершины 0:

Введите номер начальной вершины: 0
Расстояние от вершины 0 до 5  =  7     Путь: 0 4 5
Расстояние от вершины 0 до 4  =  4     Путь: 0 4
Расстояние от вершины 0 до 3  =  12    Путь: 0 4 3
Расстояние от вершины 0 до 2  =  12    Путь: 0 1 2
Расстояние от вершины 0 до 1  =  7     Путь: 0 1
Расстояние от вершины 0 до 0  =  0     Путь: 0
